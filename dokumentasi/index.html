Prepare

Before installing let's first prepare the data needed:

  1. Persiapkan API Key Raja Ongkir: Sebelum menggunakan layanan Raja Ongkir, Anda perlu mempersiapkan kunci API yang diperlukan untuk mengautentikasi permintaan Anda. Berikut adalah langkah-langkah untuk mendapatkan kunci API:
    • Langkah 1: Masuk ke akun Anda di Raja Ongkir Dashboard.
    • Langkah 2: Navigasi ke bagian "Pengaturan Akun" atau "API Key" di dashboard Anda.
    • Langkah 3: Klik tombol "Buat API Key" atau sejenisnya.
    • Langkah 4: Salin dan simpan kunci API yang dihasilkan. Pastikan untuk merahasiakannya dan tidak membagikannya dengan orang lain.
  2. Persiapkan API Key Midtrans: Untuk mengintegrasikan pembayaran dengan Midtrans, Anda perlu memiliki kunci API yang akan digunakan untuk berkomunikasi dengan sistem pembayaran Midtrans. Berikut adalah cara mendapatkan kunci API Midtrans:
    • Langkah 1: Login ke akun Midtrans Anda di Midtrans Dashboard.
    • Langkah 2: Navigasi ke bagian "Settings" atau "API Configuration" di dashboard Anda.
    • Langkah 3: Di bagian "API Access Key", klik tombol "Generate New Key".
    • Langkah 4: Salin dan simpan kunci API yang dihasilkan. Ingatlah untuk menjaga kerahasiaan kunci ini.
  3. Persiapkan Midtrans Authorization: Setiap permintaan yang Anda kirimkan ke Midtrans API perlu diotorisasi dengan menggunakan kunci enkripsi. Berikut adalah link untuk membuat otorisasi: di sini
    ganti "abc" dengan key midtrans milik Anda, di sana contoh jika saya masukan abc akan menghasilkan "Basic YWJjOg=="

Configuration

Berikut langkah - langkah konfigurasi:

  1. Buat database di akun hosting Anda lalu import file eone_store.sql yang disertakan di dalam folde .zip
  2. Edit File .ENV sesuaikan dengan data koneksi Anda
  3. Buat file .htaccess
    • Options +FollowSymLinks -Indexes IndexIgnore * DirectoryIndex RewriteEngine on # Simple URL redirect: RewriteRule ^(.*)$ http://127.0.0.1:8090/$1 [P]

Installation

Berikut langkah - langkah installasi:

  1. jika langkah-langkah dari Prepare s/d Configuration sudah selesai, silahkan akses domain Anda yang sudah diatur di file .env,http://example.com/ maka akan auto diarahkan ke halaman setup seperti gambar berikut :
    2. masukan licence key yang telah disertakan pada file licence.txt
  2. jika sudah selesai maka akan diarahkan ke halaman auth, silahkan login dengan akun admin yang Anda atur di halaman setup
  3. Atur pengaturan di menu Settings untuk memasukan logo dll.
